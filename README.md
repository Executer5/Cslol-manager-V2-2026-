# 🛠️ CSLolV2

A modern and lightweight custom skin manager for League of Legends. CSLolV2 allows players to load, manage, and organize custom skins, VFX, models, sounds, and other cosmetic modifications with a fast and simple interface designed for performance and usability.

---

# ✨ Features

- ⚡ Fast and lightweight performance
- 📦 Support for custom skin mods
- 🛠️ Easy mod importing and management
- 📂 Profile system for different mod setups
- 🔍 Search and filter support
- 🎨 Clean modern interface
- 🔄 Automatic loading system
- 📁 Organized mod structure
- 🚀 Optimized startup speed
- 🧩 Expandable architecture for future updates

---

# 📂 Project Structure

```bash
CSLolV2/
│
├── assets/                 # Images, icons and UI resources
├── configs/                # Configuration and settings files
├── core/                   # Core application systems
├── loaders/                # Mod loading systems
├── modules/                # Extra modules and extensions
├── profiles/               # Saved user profiles
├── logs/                   # Debug and runtime logs
├── temp/                   # Temporary generated files
├── ui/                     # User interface components
├── utils/                  # Utility/helper functions
│
├── main.cpp                # Main application entry
├── launcher.cpp            # Launcher system
├── manager.cpp             # Mod manager backend
├── README.md               # Documentation
├── LICENSE                 # License file
└── config.json             # Main configuration file
```

# 🚀 Getting Started

## Requirements

- Windows 10 / 11
- League of Legends installed
- Visual C++ Redistributables

---

# 📥 Installation

```bash
git clone YOUR_REPOSITORY_URL
cd CSLolV2
```

---

# ▶️ Usage

Launch the application:

```bash
CSLolV2.exe
```

Select your League of Legends folder and start managing your custom skins easily.

---

# 🛠️ Mod Support

CSLolV2 supports:

- Custom VFX
- Sound replacements
- HUD modifications
- Map changes
- UI modifications
